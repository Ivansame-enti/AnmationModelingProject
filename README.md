Explicación Ej2:
Se ha utilizado un simple Lerp implementado por nosotros para realizar la animación, pues al tratarse todas las rotaciones simplemente en un eje, se puede tratar como un problema 2D, y, por tanto, solo modificar el ángulo de rotación, evitando así tener que hacer interpolaciones con los quaternions.
